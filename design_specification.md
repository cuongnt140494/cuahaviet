## 10. Components UI Cần Thiết Kế

| Component | Mô tả | States |
|-----------|-------|--------|
| **Button** | Primary, Secondary, Ghost, Danger | Default, Hover, Active, Disabled |
| **Input** | Text, Password, Textarea, Select | Default, Focus, Error, Disabled |
| **Card** | Product Card, Category Card, Post Card | Default, Hover |
| **Badge** | New, Hot, Sale | - |
| **Modal** | Confirm, Form, Image Lightbox | - |
| **Toast** | Success, Error, Warning, Info | - |
| **Table** | DataTable với pagination, sorting | - |
| **Tabs** | Horizontal tabs | Active, Inactive |
| **Breadcrumb** | Navigation trail | - |
| **Pagination** | Page numbers + Prev/Next | - |
| **Skeleton** | Loading placeholder | - |
| **Empty State** | Khi không có data | - |
